# Email Analysis and Classification for SPAM Detection

## Project Description

The goal of this project is twofold:

1. Classify received emails as SPAM or NOT SPAM.
2. Analyze SPAM emails to identify:
  - The main topics.
  - The semantic distance between topics.
  - The organizations mentioned in NOT SPAM emails.
    
## Objective
This project provides a pipeline that combines classification, topic modeling, semantics, and entity recognition (NER).
